This project created with React Native Community CLI

# Getting Started

#### JAVA-JDK Version 17 (Recommend)
#### node.js Version >=18
#### Yarn Version 3.6.4

## Step 1: Install Modules

```bash
# using yarn
yarn install
```

## Step 2: Start your Application
### For Android

```bash
# using Yarn
yarn android

# using npx
npx react-native run-android
```

### For iOS

```bash
# using Yarn
yarn ios

# using npx
npx react-native run-ios
```
